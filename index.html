Mongoose tutorial

Mongoose setup

Step 1 : First be sure you have MongoDB and Node.js installed.

Step 2 : Next install Mongoose from the command line using npm:

npm install mongoose

Step 3 : Import the mongoose module.

:-const mongoose = require('mongoose');

Step 4 : Connection creation and creating a new database.

:-const mongoose = require('mongoose');
main().catch(err => console.log(err)).then(()=> console.log("connection succesfull.."));
async function main() {
await mongoose.connect('mongodb://localhost:27017/dbname');
}

Step 5 : Defines the structure of the documents with the help of schema.

:- const playlistSchema=new mongoose.Schema({
   name:{
   type: String,
   required: true
      },
  ctype:String,
  videos:Number,
  author:String,
  active:Boolean,
  date:{
           type:Date,
           default:Date.now
      }
})

Step 6 : collection creation with the help of model

const Playlist = new mongoose.model("Playlist",playlistSchema);

Step 7 : Create and insert documents

(1) Insert one document

Playlist({
    name:"React JS",
    ctype:"Front End",
    videos:80,
    author:"Thapa Technical",
    active: true
        }).save()

(2) Insert many document

const htmlPlaylist=new Playlist({
   name:"html",
   ctype:"Front End",
   videos:80,
   author:"Thapa Technical",
   active: true
        })
const nodePlaylist=new Playlist({
   name:"node JS",

      })
const mongoDBPlaylist=new Playlist({
